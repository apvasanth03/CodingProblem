# Reverse Linked List

[LeetCode Question Link](https://leetcode.com/explore/interview/card/top-interview-questions-easy/93/linked-list/560/)

[Solution](https://github.com/apvasanth03/CodingProblem/blob/master/src/main/kotlin/com/vasanth/codingproblem/leetcode/topquestions/easy/linkedlist/ReverseLinkedList.kt)

Reverse a singly linked list.

**Example:**

```
Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL
```

**Follow up:**

A linked list can be reversed either iteratively or recursively. Could you implement both?
